# Tech Stack

Decided at the start of each project based on scope and complexity. Use the decision guide below, then document the chosen stack.

---

## Decision Guide

| Scenario | Recommended Stack |
|----------|------------------|
| Simple brochure site, no CMS needed | Plain HTML/CSS/JS or Astro |
| Brochure site, client needs to update content | Astro + a headless CMS (e.g. Sanity, Decap) |
| Multi-page site with dynamic features (forms, booking) | Next.js |
| E-commerce | Next.js + Stripe / Shopify |
| Anything with a database or user accounts | Next.js + Supabase |

**Rule of thumb:** choose the simplest stack that covers all requirements. Don't over-engineer.
